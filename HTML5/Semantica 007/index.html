A morte das tags.

As tags tornam-se obsoletas e outras surgem no lugar. Um exemplo é o negrito. Existe a tag que deixa o termo em negrito, b do inglês bold. Entretanto, está sendo substituída pela tag onde tem o sentido de forte.Isso acontece para trazer mais significado e menos estilo para o HTML, os estilos ficam com as CSS.

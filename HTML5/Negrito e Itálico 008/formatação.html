Marca Texto

Nesta frase, temos um texto destacado com marca texto usando a tag <mark>. Para mudar as cores desse marca texto, apenas com as CSS. Uma das formas é usando o parametro style dentro da tag <mark>. Porém, desta forma altera apenas o <mark> atual, para alterar a cor padrão para padronizar todas as cores padrão do site, falando em marcador de texto, precisa ser feito o style background- color abaixo do título.

Texto Deletado

Para deletar um texto, mas ainda deixar a informação ali para conhecimento de como era, deve-se apenas usar a tag <strike>, com o HTML5, usa-se a tag <del>. Inclusive, foi a tag usada para este parágrafo.

Texto Inserido

O texto inserido é o exato oposto do texto deletado que vimos anteriormente. Neste caso, se colocamos um texto qualquer dentro de <ins> e </ins>, estamos querendo informar que o texto sublinhado foi inserido na nova atualização ou que se deve prestar a atenção nessa parte.

Texto Sobrescrito e Texto Subscrito

Sobrescrito

Para escrever x2 basta colocar o que vai ficar na parte superior da letra entre a tag <sup> e dentro da tag <small> para ficar pequenininho o número.

Subscrito

Para escrever na base da letra como em uma fómula química H2O, coloca-se entre a tag <sub> e dentro da tag <small> para ficar pequenininho o número.

Código Fonte

Existe a tag <code> da HTML onde se pode delimitar o código.

Exemplo:

        nr = int(input('Digite um número.'))
            for c in range(0, 10):
                tb = c * nr
                print('{} X {} = {}'.format(c, tb, nr))

Citações

Para citações no meio da frase, basta colocar o trecho entre a tag <q>. Desta forma, o trecho ficará entre áspas.

Para citações em bloco, usa-se outra tag, a <blockquote>.

Exemplo:

Em seu livro, O Mestre da Sensibilidade, Augusto Cury conclui poeticamente, sobre Jesus, que:

Ele tinha todos os motivos para ter depresão e ansiedade, mas nunca alguém foi tão feliz e livre no território da emoção.

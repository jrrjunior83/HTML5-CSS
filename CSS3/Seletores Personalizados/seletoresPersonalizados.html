HTML Básico

Ao se criar o id=principal abre a possibilidade de o primeiro H1 ter uma configuração diferente dos outros H1s. Porém isso só pode ser feito uma vez. Para alterar mais ítens, precisa-se criar o class, '.' em CSS. e nele fazer variadas configurações.

HTML Intemediário

Com a tag <span> pode se selecionar parte de qualquer texto e aplicar o class que desejar, seja básico, intermediário ou avançado. Também é possível usar mais de uma classe para um mesmo elemento. Por exemplo, a class destaque será aplicada juntamente com as classes; básico, intermediário ou avançado. .Foi apenas, dentro de class de cada um dar espaço e escrever o nome da classe destaque, sem usar vígula ou qualquer pontuação. Também é possível incluir class junto do id, entretanto o id sobrepoem o class.
